##  주요 기능

<div className="space-y-6 mt-4">

###  게시글 목록 조회

![메인 페이지](/img/main.png)

<div className="text-base text-gray-800">
  클라이언트에서 GET /api/get 요청을 보내면 서버는 MySQL에서 전체 게시글 데이터를 불러와 JSON 형식으로 응답합니다.  
  React에서는 이 데이터를 Axios로 받아 리스트 형태로 화면에 출력합니다.  
  각 게시글 항목은 클릭 시 해당 글 상세 페이지로 이동할 수 있습니다.
</div>

---

###  글 작성

![글 작성 페이지](/img/min.png)

<div className="text-base text-gray-800">
  사용자가 제목과 내용을 입력한 뒤, POST /api/insert 요청을 통해 서버에 데이터를 전송합니다.  
  서버에서는 이 데이터를 MySQL의 kimawsdb 테이블에 INSERT 쿼리로 저장합니다.  
  작성일자 또한 함께 저장되며, 저장이 완료되면 클라이언트는 자동으로 목록 페이지로 이동합니다.
</div>

---

###  글 수정

![글 수정 페이지](/img/sujun.png)

<div className="text-base text-gray-800">
  글 목록에서 게시글을 선택하면 수정 폼으로 이동하며, 기존 제목과 내용이 입력된 상태로 표시됩니다.  
  수정 후, PUT /api/update 요청을 통해 서버에 수정된 데이터와 idx를 전송합니다.  
</div>

---

###  글 삭제

![글 삭제 페이지](/img/dlet.png)

<div className="text-base text-gray-800">
  삭제 버튼 클릭 시 JavaScript의 window.confirm으로 사용자에게 확인 메시지를 띄운 후,  
  DELETE /api/delete/:idx 요청을 통해 해당 게시글을 서버에 삭제 요청합니다.  
  서버는 전달받은 idx로 MySQL에서 해당 idx를 제거하며, 삭제 후 클라이언트는 자동으로 목록을 새로고침합니다.
</div>

</div>
